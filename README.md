减法-莱迪·克洛茨
```
学会给生活做减法
	给问题找答案
		为什么 —> 如何做
	拥有的越多，要做的事情也越多

减法的力量
	概念
		只要资源对很多人都有用且不属于任何一个人，自私的行为就会占上风
		之所以能成就这一切，正是因为抓住了其他人都没有看到的机会
	改变方式
		加法
		减法
	误区
		如果我们在错误的知识基础上进行建构，就无异于试图去加固遭地震破坏的高速公路
	核心
		减法，有时是少做一件事让自己有更多时间精力做更重要的事，有时意味着要做的更多想的更多

看见“更多”
	概念
		人们很容易把减少当成一种损失
		就像具有获取物品的本能一样，对于数量的本能当然是求“多”，而不是求“少”
	“少”是一种更好的选择
		“少”的真谛
			减法也是改变事物的一种方式
		乐高、实验室及其他
			人们选择加法并不是因为加法比减法更省力，而是减法考虑的更多更麻烦
			选择加法，或许是因为我们更喜欢自己制作的东西
			选择加法，或许是因为删减东西就意味着要承认之前添加的属于沉没成本
			或许是因为失去比收获更令人不安
			假如根本就没有考虑到减法，那就说明缺乏全面的认识，也势必会造成错误和疏漏
		人们总喜欢做加法
			并不只是主观上喜欢加法，而是最开始就没有想到减法
		减法被忽略的背后
			忽视减法有着深刻且复杂的先天和后天因素
	人类做加法的本能
		展示自身能力的本性
			无论减法的行为多么有益，都不可能像加法一样为所做的事情留下那么多证据
			生物特性让我们即使想到了一些有利的减法，也不愿意去做
		天生获取物品的本能
			东西选取太多而减少不了
			获取物品的本能使我们倾向于要更多的东西
			对做加法的本能了解得越多，就越能更好地抵制这种本能
		感知相对数量的本能
			依靠直觉，而不是算术，就会比较的数量越接近，给出正确答案的可能性就越小
			感觉到的变化取决于初始状态
			一到二的变化就要大于八到九的变化。数量越大，感知到的新变化所带来的差异就越小
			准确度不取决于最终的数量，而是取决于要获得最终数量所需的增减量
			对相对数量的直觉让我们很难准确地想象减法的结果，而且还是想象减掉的比实际存在的还要多，那真是“不可思议”
			数量越少，减少反而会带来更大的比例
		对于数量的固定思维模式与直觉是如何影响减法的
			直觉思维
				数量越少差异越大，数量越大差异越少
			固定思维
				具有一定数量的某种物品
			距离模式
				在脑海中形成一根数轴，把每个数字放在数轴相应的位置上
		在加减平衡中进化
			祖先在敲打石头制作武器就是运用减法
			现代雕刻艺术就是运用减法
			本能会做加法，但是可以有意识的做减法
	加法带来文明，文明带来累积
		加法定义了最早的文明
			耕种 —> 定居 —> 建造纪念性建筑 —> 催生文明
			建造纪念性建筑 —> 催生文明 —> 耕种 —> 定居
			物质文化
				人类可以根据服装、饰品等将自己所接触到的人进行分类，而无须了解成千上万的个体
				使素未谋面的人知道该如何与对方进行交往
				敢于冒险的猎人可以穿猛犸象皮，而胆小谨慎的猎人只能穿兔皮。人们不再需要记住团队中谁喜欢冒险、谁不喜欢冒险，只需通过服装就能做出判断
				只有在每天必须的事情上可以少花时间，不用为其烦恼（做减法），才能有更多的时间做自己想做的事（做加法）
		不同世界观对“少”的认知
			冲突
				一类是那些根据自己的能力、价值观和态度来定义自己的人（独立型）
				另一类是那些根据自己的人际关系、社会角色和群体归属来定义自己的人（依存型）
		4个区域以及4个广场
			少搭一块积木意味着付出更少，但通过减法来做改变却需要更多的思考
		既考虑加法，也考虑减法
			找到缺失的部分，从而选择用加法还是减法
			加减法二选一还是共存
			如果一个想法与另一个想法相矛盾，那么其中一个想法就必须被否定
			问题不在于“我们应该增加还是减少”，而在于“我们该如何进行合理的增减”
	时间、金钱与崇尚加法的现代社会
		重复的设想
			不断重复“更多”，强调加法的使用
			“更多”能让世界变得更好，但并不意味着“更少”做不到
		我们为什么越活越累
			在追寻物质财富的过程中，耗尽了自己最宝贵的资源，那就是时间。今后，我们的时间只会越来越少
		给产品做减法
			待办事项越少，生活满意度就会越高
		把关注点放在人的身上
			既关注工具，也关注人
		同义词
			加法
				增加、附加、增大、加强、覆盖、连接、接合、多和强化
			减法
				减少、分离、不含、更少、限制、不、去除、简化和无

分享“更少”
	概念
		减法思维，不是要求我们简单地摒除冗余的信息，而是教会我们在日常生活中，洞察真相，看得更透彻，活得更轻松
	思维反转：找到减法，分享减法
		满意度达成后的“更少”
			在足够好的时候停一下，可以让我们不浪费精力。但如果不够慎重，我们还可能错失进一步做减法的机会
			你有东西才能使用减法
			如果此前建造的东西没有损坏，就不必修补
		引人注目的“更少”
			在写论文、搞设计或者做计算的时候，不愿意删除任何一个部分。这倒不是因为每个部分都对主题和内容有帮助，而是因为我们想让别人看到我们所做的工作
			要让你的减法做得引人注目，通常是需要寻求帮助的
		引人注目的“更少”（在专业编辑帮助下完成）
			我们可以通过别人的帮助获益，最理想的帮手就是对我们先前工作没有多少依赖性的人，或者是擅长做减法的专业人士
		减至激发出喜悦为止
			要找到我们真正需要的，最好的办法就是把那些我们不需要的扔掉
		学会对减法进行转换
			心理效价是一个事件、物体或想法对人的内在吸引力（积极效价）或带来的厌恶感（消极效价）
		动词
			揭开
			清理
			切割
		损失厌恶
			赢得100美元的满足感比不上输掉100美元的失望，即对损失的反应强于对收益的反应
			比如，试穿衣服的感觉越好，提供额外的服务越多，不买就会感觉亏了
			减法是一种改进，而且这种改进即使在数量上有所减少，也是一种进步而不是损失
			在考虑一个项目的时候，我经常想的是项目已有的存在，然后再加上对其进行改造和转变的内容
	减法清单：用减法改变系统
		看清系统本身
			我们可能会盲目地增加一些好的东西，而没有理解这些东西之间存在何种关联
			如果想要去除不好的东西，我们首先得发现它，并且认定它确实不好
			考虑的东西越多，想的也就越复杂
		忽略的才是更好的
			如果做改变时忽略了某个类型的所有选择，其实已经够糟糕了
			行为上的改变
				坏方法
					就是做加法，无论是对好的行为增加激励还是对坏的行为加大惩罚，这种加法都会加剧系统内部的紧张程度
					承诺并不能让自己抵制住诱惑。事实上，一旦没能抵制住诱惑，这样的承诺就会让自己更加沮丧
				好方法
					把诱惑放在看不到接触不到的地方，这样才能抵制住诱惑
				变化
					在做减法时，也是在重新思考
		拥抱复杂，追求本质
			要对复杂系统做决策，精简的清单比面面俱到更有效
			减少不必要的细节，找到重要的东西
		删去细节，把握本质
			同样的问题在不同的场景，有着不同的解决方法
			解决多个问题还是中途加入了新的问题，都是要有轻重缓急（排列顺序）
		少即是多
			在早期发现问题，总比后期发现好
			 重复利用删除的内容
			在改进系统前删减细节
			 坚持让“少”看得见，实现质的飞跃
	“更少”的遗产：人类世的减法
		用更少的资源做更多事情
			做加法还是减法其实就是找到平衡点
		减少，再利用，再循环
			用制造问题的思维模式去面对问题，那简直就是最大的讽刺
			坚持做让人看得见的减法
		学会选择，留下机遇
			物资是有限的，知识是用之不竭的
	从信息到智慧：减法实现富足人生
		删减信息，积累智慧
			在一个方面进行信息处理，就意味着无法在另一个方面进行信息处理
			致贫的不是错误的决定，而是身处贫穷的认知导致了错误的决定
			有太多事情要做，便会想着偷工减料

		过滤信息，实现突破
			对学过的知识进行删减，提炼有价值的知识
			反复思考重要的事情，从而加深印象
			多思考，多实践
		学会去除错误想法

			不去除错误的想法，新的想法也可能被扭曲
		类比：对思想做减法
			找到相同的例子进行对比，发现优缺点，从而打破错误观点
			忘掉一些细节，从更清晰的视角看待
```
